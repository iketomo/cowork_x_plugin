# /line-message

LINE公式アカウントの「関心引き出しメッセージ」を作成します。
行動経済学・心理学の知見に基づき、1画面（150〜300文字）で「興味ある」ボタンをタップさせるメッセージを3パターン提案します。

## 実行手順
1. `line-message` スキルを Skill ツールで呼び出す
2. $ARGUMENTS が指定されている場合は、その内容を配信目的・ターゲット・訴求内容として渡す

## 使用例
- `/line-message` - 対話的に目的・ターゲットを確認してからメッセージ作成
- `/line-message セミナー集客 AIマーケティング入門 3/25開催 定員30名` - 指定した条件でメッセージ作成
