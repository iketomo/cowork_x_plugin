# cowork-manager

Coworkプラグインの**作成・カスタマイズ・管理**を支援するプラグイン。

3つのスキルと3つのコマンドがそれぞれ専用のサブエージェントに処理を委譲する軽量アーキテクチャ。

## コンポーネント

### コマンド

| コマンド | 引数 | 概要 |
|---|---|---|
| `/create-plugin` | `[plugin-name]` | プラグイン新規作成を起動 |
| `/customize-plugin` | `[plugin-name]` | 既存プラグインのカスタマイズを起動 |
| `/list-plugins` | `[plugin-name]` | インストール済みプラグインの一覧・詳細を表示 |

### スキル

| スキル名 | トリガーフレーズ例 | 委譲先エージェント |
|---|---|---|
| `create-plugin` | 「プラグインを作って」「新しいプラグインを作成」 | `plugin-creator` |
| `customize-plugin` | 「プラグインをカスタマイズ」「プラグインの設定を変えたい」 | `plugin-customizer` |
| `list-plugins` | 「プラグイン一覧」「どんなプラグインが入ってる」 | `plugin-lister` |

### エージェント

| エージェント名 | 役割 |
|---|---|
| `plugin-creator` | プラグイン新規作成（5フェーズの対話ワークフロー） |
| `plugin-customizer` | 既存プラグインのカスタマイズ・コネクタ設定 |
| `plugin-lister` | インストール済みプラグインの一覧表示・内容要約 |

## 使い方

### プラグインを新規作成する

「新しいプラグインを作って」と話しかけると、`plugin-creator` エージェントが起動します。ディスカバリーから `.plugin` ファイルの生成まで、対話を通じて進めます。

### 既存プラグインをカスタマイズする

「〇〇プラグインをカスタマイズしたい」と話しかけると、`plugin-customizer` エージェントがプラグインを特定し、組織のツールやワークフローに合わせて調整します。

### インストール済みプラグインを確認する

「どんなプラグインが入ってる？」と話しかけると、`plugin-lister` エージェントが一覧と各機能の概要を整理して表示します。

## セットアップ

環境変数や追加設定は不要です。Coworkデスクトップアプリにインストールするだけで動作します。

> カスタマイズ機能はCoworkデスクトップアプリのローカル環境でのみ利用可能です。
