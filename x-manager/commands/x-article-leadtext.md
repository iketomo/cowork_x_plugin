# X記事紹介本文生成

`x-article-leadtext` スキルに従い、X記事の魅力紹介ポスト本文を生成する。

## 実行手順

### Step 1: 記事情報の取得

- `$ARGUMENTS` に記事URL・タイトル・要約が指定されていればそれを使用
- URLのみの場合は、ユーザーにタイトル・要約・キーポイントの補足を依頼（Web検索は任意）
- 何も指定がない場合は、記事情報の入力をユーザーに依頼

### Step 2: スキル実行

`x-manager/skills/x-article-leadtext/SKILL.md` の手順に従う：

1. 記事情報を整理（タイトル・要約・ターゲット・キーポイント）
2. ジャンルに合ったパターン1-2個を選択
3. AIDA-Xフレームワークでドラフト作成
4. X最適化チェック
5. バリエーション2-3案を出力

### Step 3: 結果をユーザーに提示

各案を「パターン名・完成文案・文字数・補足メモ」の形式で提示。  
修正や追加パターンの要望があれば再生成する。
